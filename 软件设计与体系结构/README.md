现假设要实现一个医院系统的部分功能<br>

  病人就诊：病人看病需要经过挂号（选择诊疗科目、选择医生），医生确诊，付费，医生治疗，结束治疗5个步骤，不同的步骤，不同的医院有不同的做法，请用模板方法模式抽象出这个病人就诊步骤，并能方便扩展。<br>
